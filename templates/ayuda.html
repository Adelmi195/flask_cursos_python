Guía de Uso del Programa

  • Cargar CSV: Selecciona un archivo CSV con tus datos
    • Primera fila debe contener los nombres de las columnas
    • Solo se procesarán columnas numéricas
    • Valores faltantes serán eliminados automáticamente
  • Simular Datos: Genera datos sintéticos para pruebas
    • Puedes especificar el número de muestras
    • Se generarán datos con clusters conocidos
    • Útil para aprender y probar el algoritmo

  • Método de Enlace:
    • Ward: Recomendado para la mayoría de casos
    • Complete: Para clusters compactos
    • Average: Método balanceado
    • Single: Para clusters alargados
  • Métrica de Distancia:
    • Euclidiana: La más común
    • Manhattan: Menos sensible a outliers
    • Coseno: Para datos de alta dimensionalidad
  • Número de Clusters: Especifica cuántos grupos quieres
  • Normalizar Datos: Recomendado para la mayoría de casos

  • Presiona "Ejecutar Análisis" para procesar los datos
  • Los resultados aparecerán en diferentes pestañas:
    • Dendrograma: Muestra la jerarquía de clusters
    • Scatter Plot: Visualización de los clusters (2D)
    • Estadísticas: Información detallada de cada cluster

  • Dendrograma: Observa la altura de las fusiones
  • Scatter Plot: Verifica la separación visual de los clusters
  • Estadísticas: Analiza las características de cada cluster
  • Información: Revisa métricas de calidad

  • "Error al cargar CSV": Verifica que el archivo tenga columnas numéricas
  • "Datos insuficientes": Necesitas al menos 2 filas de datos
  • "Error en análisis": Revisa los parámetros seleccionados
  • "Visualización vacía": Ejecuta primero el análisis

  • Siempre normaliza los datos cuando uses distancia euclidiana
  • Prueba diferentes métodos para comparar resultados
  • El método Ward funciona bien con datos bien escalados
  • Usa el dendrograma para decidir el número de clusters
  • Valida los resultados con tu conocimiento del dominio

Archivo separado por comas

Primera fila: nombres de columnas

Solo columnas numéricas serán procesadas

Ejemplo:

Variable1,Variable2,Variable3
1.5,2.3,4.1
2.1,1.8,3.9
...

  • Ctrl+O: Abrir archivo CSV
  • Ctrl+S: Guardar resultados
  • Ctrl+R: Ejecutar análisis
  • Ctrl+C: Limpiar análisis
  • F1: Mostrar esta ayuda
